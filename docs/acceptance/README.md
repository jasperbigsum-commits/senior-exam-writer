# Acceptance Artifacts

This directory contains the final small-sample acceptance artifacts for `senior-exam-writer`.

## Files

- `senior_exam_writer_actual_question_package.docx`: human-readable Word question package.
- `senior_exam_writer_actual_question_package.json`: structured question package with items, answers, evidence references, and review metadata.
- `senior_exam_writer_artifact_verification.md`: human-readable verification report for package, local RAG knowledge bases, and prepare artifacts.
- `senior_exam_writer_artifact_verification.json`: structured verification report.

## Scope

These artifacts were generated from small mock materials to verify the skill workflow, evidence traceability, sqlite-vec local RAG persistence, and final package shape. They are acceptance fixtures, not official exam materials or real historical exam papers.
